## 🛠️ Common Issues & Fixes

### ❓ "App not installed" error on Android
**Fix:** Fully uninstall the original Roblox app before installing the Hydrogen APK. Go
to **Settings → Apps → Roblox → Uninstall**, reboot, then install the
**hydrogen executor apk android** build fresh.

### ❓ Key verification won't complete
**Fix:** Switch networks (try mobile data if on Wi-Fi, or vice versa). Disable any active
VPN before attempting verification. If the issue persists, check **#key-support** in the
**hydrogen executor discord server** — CDN outages occasionally affect key delivery.

### ❓ macOS shows "App is damaged" on launch
**Fix:** Run the following in Terminal to remove the quarantine flag:
```bash
xattr -cr /Applications/HydrogenExecutor.app
```
This is a standard macOS security behavior for downloaded apps and is not an indicator
of any actual issue with the **hydrogen script executor mac** build.

### ❓ Zip file won't extract on macOS
**Fix:** Use **The Unarchiver** (free on the Mac App Store) instead of the built-in
Archive Utility. Some versions of macOS Archive Utility incorrectly flag the
**hydrogen executor zip download** as corrupt when it is not.

### ❓ Floating icon not showing on Android
**Fix:** Go to **Settings → Apps → Hydrogen → Permissions** and enable
**"Display over other apps"**. On MIUI or HyperOS devices, additionally enable
**"Background pop-up windows"** in the app's special permissions section.

### ❓ "Failed to attach" on Windows PC
**Fix:** Right-click the Hydrogen executable and choose **Run as Administrator**. Also
add the Hydrogen installation directory to Windows Defender's exclusions list — the
injection library is frequently flagged as a false positive.

### ❓ Scripts from the hub return errors or don't execute
**Fix:** Ensure you're running the latest version of Hydrogen. Scripts in the hub are
maintained for the current Roblox client, and older executor builds may have
incompatible API environments. Check **#script-status** in the Discord for game-specific
compatibility notes.
